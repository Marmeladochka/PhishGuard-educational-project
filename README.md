# PhishGuard

PhishGuard -- это образовательный прототип для обнаружения фишинговых сообщений с использованием
анализа на основе правил в сочетании с облегченными методами машинного обучения.

Этот проект создан исключительно для образовательных целей.

### Особенности
- Анализ текста для выявления фишинговых индикаторов
- Гибридный подход: правила + машинное обучение
- Консольный интерфейс
- Объясняемые результаты

### Необходимые инструменты
- Python 3.10+
- scikit-learn
- numpy, scipy
- colorama

### Запуск
```bash
pip install -r requirements.txt
python script.py
